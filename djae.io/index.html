About

This is my first time building a pure CSS website using SASS. I followed an master class CSS course on Udemy. A lot of features offered here is only supported in few browsers. Namely, clip-path, shape-outside and object-fit doesn't work IE nor MS Edge For full experience, you should use Google Chrome.

Accomplishment

I learned features of SASS: mixins, variables, using mixins to better organize media queries, modern CSS techniques (clip-path, transform, animation, keyframes, background-image: linear gradient url()). I also learned about nodemon utility that automatically restarts server upon document changes.
