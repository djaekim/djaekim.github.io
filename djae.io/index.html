Open Tennis Website

Grid: Rows and Coloumns

  • Rows are block level - takes up entire width of element its in
  • Only Col are direct child of Row and all the contents should go inside col
  • Cols horizontally aligns in grid. The Cols have 15px left/right padding to evenly space grid. Thus,row has -15px left/right padding to compensate the first and last col..
  • Rows are divded into 12 equal parts. When we put col inside rows, we need to specify number of parts the col is going to take up
  • If col parts exceeeds 12, it wraps to the next line

    • Screen size

    • xs - Smartphones / extra small
    • sm - tablets / small
    • md - laptop / medium
    • lg - large high resolution screens / large

Clearfix

  • when a col has much more content, thus height then the ones after it, the layout breaks (may pile up). Thus, we can use clearfix. It forces all columns after it move to the new line

Offset

  • To create margin on the left of the coloumn we use offset
  • Ex: col-md-6 col-md-offet-3

    • by default, without offset, any excess space right of coloumns remain empty. So, by using offset, we could push it to the right

Push and Pull

  • reorder columns depending on screen size.
  • Push moves coloumns to right, pull moves coloumns to left
  • it uses position relative thus doesn't effect other columns
  • Ex: col-xs-4 col-md-push-8

    • On bigger screen it coloumns will have 8 space on the right, but on the smaller screen, it will have 4 space on left..
